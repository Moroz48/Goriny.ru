# Быстрая установка Ngrok

## Шаг 1: Регистрация

1. Перейдите на https://ngrok.com/
2. Нажмите "Sign up" (регистрация)
3. Зарегистрируйтесь (можно через Google/GitHub)

## Шаг 2: Скачивание

1. После регистрации перейдите в Dashboard
2. Нажмите "Download" → выберите Windows
3. Скачайте файл `ngrok.zip`

## Шаг 3: Установка

### Вариант A: В папку проекта (Проще)
1. Распакуйте `ngrok.exe` в папку `E:\saut`
2. Запустите `start-ngrok.bat`

### Вариант B: В системный PATH (Для всех проектов)
1. Распакуйте `ngrok.exe` в папку (например `C:\ngrok\`)
2. Добавьте папку в PATH:
   - Win + R → `sysdm.cpl` → Enter
   - Вкладка "Дополнительно" → "Переменные среды"
   - В "Системные переменные" найдите "Path" → "Изменить"
   - Добавьте путь к папке с ngrok.exe
   - OK → OK

## Шаг 4: Авторизация (Один раз)

1. Откройте командную строку
2. Выполните:
   ```
   ngrok config add-authtoken ВАШ_ТОКЕН
   ```
   (Токен можно найти в Dashboard на ngrok.com)

## Шаг 5: Использование

1. Запустите локальный сервер: `start-server.bat`
2. В новом окне запустите: `start-ngrok.bat` или `ngrok http 8000`
3. Скопируйте ссылку вида `https://xxxxx.ngrok.io`
4. Отправьте друзьям!

## Альтернатива без установки:

Можно использовать онлайн версию через браузер:
- https://ngrok.com/product/secure-tunnels
- Но проще установить приложение
